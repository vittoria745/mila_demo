# Mila 启智润心 - 儿童心理健康陪伴应用

## 项目简介
Mila 启智润心是一款专为儿童设计的心理健康陪伴Web应用，采用吉卜力水彩美术风格。通过AI情感陪伴、创意表达、情绪管理等功能，帮助孩子们建立积极的心理状态。

## 核心功能模块
- 🌈 **彩虹花园** — 情绪记录与心情打卡
- 🎨 **艺术派** — 绘画与音乐创意表达
- 📝 **创意工坊** — 笔记本与手工创作
- 📖 **部落格** — 日记与博客空间
- 💬 **Mila聊天** — AI情感陪伴对话
- 🏘️ **Mila小镇** — 交互式探索地图（9座特色建筑）

## 如何体验
1. 解压本压缩包到任意文件夹
2. 双击打开 `mila_entrydemo.html`
3. 所有图片资源均在同一目录下，无需额外配置
4. 推荐使用 Chrome / Edge / Safari 浏览器

## 体验入口
- **孩子端**: 直接打开 HTML 即可体验完整功能
- **监护人端**: 在任意页面点击隐藏的「监护人入口」

## 技术说明
- 纯前端实现，单HTML文件应用
- 所有数据存储在浏览器 localStorage 中
- 无需后端服务器，离线可运行
- 响应式设计，支持手机端和桌面端
